Vulnerability ID 62731

jCore 1.0 rlogin sql injection

CVSSv3 Temp ScoreCurrent Exploit Price (≈)
7.0$0-$1k

A vulnerability classified as critical was found in jCore 1.0. Affected by this vulnerability is an unknown function of the component rlogin. The manipulation with an unknown input leads to a sql injection vulnerability. As an impact it is known to affect confidentiality, integrity, and availability.

The weakness was released 10/22/2012. The advisory is shared for download at jcore.net. This vulnerability is known as CVE-2012-4232 since 08/09/2012. The exploitation appears to be easy. The attack can be launched remotely. The exploitation doesn't need any form of authentication. The technical details are unknown and an exploit is not publicly available.

Upgrading to version 1.0 eliminates this vulnerability.

The vulnerability is also documented in the databases at SecurityFocus (BID 56102) and X-Force (79442). See 62730 for similar entries.
