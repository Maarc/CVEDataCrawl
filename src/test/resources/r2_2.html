Vulnerability ID 62832

Apache Axis2 up to 1.6.2 spoofing

CVSSv2 Temp ScoreCurrent Exploit Price
5.8$2k-$5k

A vulnerability, which was classified as critical, was found in Apache Axis2 up to 1.6.2. Affected is an unknown function. The manipulation with an unknown input leads to a spoofing vulnerability. This is going to have an impact on confidentiality, and integrity.

The weakness was disclosed 11/04/2012. This vulnerability is traded as CVE-2012-5785. It is possible to launch the attack remotely. The exploitation doesn't require any form of authentication. There are neither technical details nor an exploit publicly available.

There is no information about possible countermeasures known. It may be suggested to replace the affected object with an alternative product.

The vulnerability is also documented in the databases at SecurityFocus (BID 56408) and X-Force (79830). Similar entries are available at 62831.
